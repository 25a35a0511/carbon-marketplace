# 🌱 Carbon Marketplace

A modern web application for **carbon credit trading and sustainability tracking**.  
Users can explore environmental projects, purchase carbon credits, and contribute to reducing CO₂ emissions.

---

## ✨ Features

- 🌿 Browse verified carbon offset projects  
- 💳 Buy carbon credits (1 credit = 1 ton CO₂)  
- 📊 Real-time price calculation  
- 📈 Track project progress (credits sold vs available)  
- 🔔 Toast notifications for user actions  
- 🎨 Modern UI with responsive design  

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Styling:** CSS  
- **State Management:** React Hooks  
- **API Integration:** REST APIs
